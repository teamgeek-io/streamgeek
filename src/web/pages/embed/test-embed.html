Video Embed Test Page

1. Responsive Embed (Recommended)

This embed automatically adjusts to the container width while maintaining a 16:9 aspect ratio.

<div style="width: 100%; height: 0; padding-bottom: 56.25%; position: relative;"> <iframe src="/embed/YOUR_VIDEO_ID" style="position: absolute; top: 0; left: 0; width: 100%; height: 100%; border: none;" allowfullscreen> </iframe> </div>

2. Fixed Size Embed

This embed has a fixed size of 640x360 pixels.

<iframe src="/embed/YOUR_VIDEO_ID" width="640" height="360" style="border: none;" allowfullscreen> </iframe>

3. Fullscreen Embed

This embed takes up the full viewport height.

<iframe src="/embed/YOUR_VIDEO_ID" style="width: 100%; height: 100vh; border: none;" allowfullscreen> </iframe>

4. Custom Size Embed

This embed has a custom size of 800x450 pixels.

<iframe src="/embed/YOUR_VIDEO_ID" style="width: 800px; height: 450px; border: none;" allowfullscreen> </iframe>

5. Mobile-Friendly Test

This section simulates a mobile viewport (you can resize your browser window to test).

Testing Tips:
  • Replace YOUR_VIDEO_ID with an actual video ID from your database
  • Test on different screen sizes and devices
  • Check that the video controls work properly in the embedded context
  • Verify that fullscreen functionality works
  • Test with different browsers (Chrome, Firefox, Safari, Edge)
